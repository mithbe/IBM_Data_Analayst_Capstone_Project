# IBM_Data_Analayst_Capstone_Project
Analyzed global developer data from job postings, surveys, and APIs to identify in-demand programming skills, databases, and IDE trends. Cleaned and wrangled data, applied statistical analysis, and visualized insights with dashboards to inform tech skill demand.

Course Overview

This course guides you through a complete data analysis project, covering data collection, wrangling, exploratory analysis, visualization, and dashboard creation. The final project emphasizes presenting insights effectively.
